# 📊 Attendance Tracker

A simple and interactive **React-based Attendance Tracker** that allows users to view, filter, and analyze student attendance data.

---

## 🚀 Features

- 📋 Display list of students with attendance percentage
- ✅ Filter students by:
  - All
  - Present (≥ 75%)
  - Absent (< 75%)
- ⚠️ Highlight students with low attendance (< 75%)
- 📌 Click to select and highlight a student
- 🔄 Dynamic sorting by attendance (highest to lowest)
- ⏳ Loading and error handling for API calls

---

## 🛠️ Tech Stack

- **React.js**
- **JavaScript (ES6+)**
- **HTML & CSS**
- **Fetch API**
